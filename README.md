# WeChatOpenDevTools

### 介绍

这是一个专门为爬虫领域制作的轮子,用来快速调试PC端的微信浏览器和微信小程序.

**注意只支持 Window 平台！！！！**

注意本库只能作为学习用途, 造成的任何问题与本库开发者无关, 如侵犯到你的权益，请联系下面的作者QQ说明来意后删除

### 使用方式
WeChatAppEx.exe版本：8447
编译version.dll放到WeChat.exe目录下：C:\Program Files (x86)\Tencent\WeChat\[3.9.7.29]
重启微信即可开启
